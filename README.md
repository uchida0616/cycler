# カリキュラム外の技術  
  ・ Google Map API  
  ・ OpenWeatherMap  
  
# 実行手順  
<code>$ git clone git@github.com:uchida0616/cycler.git</code>  
<code>$ cd cycler</code>  
<code>$ bundle install</code>  
<code>$ rails db:create && rails db:migrate</code>  
<code>$ rails s</code>
