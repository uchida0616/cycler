# バージョン情報
  ・ Ruby 2.6.5  
  ・ Rails 5.2.6  
  
# 就業termの技術  
  ・ 友達つながり機能  
  ・ Ajax  
 
# カリキュラム外の技術  
  ・ Google Map API  
  ・ OpenWeatherMap  
  
# 実行手順  
<code>$ git clone git@github.com:uchida0616/cycler.git</code>  
<code>$ cd cycler</code>  
<code>$ bundle install</code>  
<code>$ rails db:create && rails db:migrate</code>  
<code>$ rails s</code>
